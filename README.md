# 📊 Financial Statement Analysis using MS Excel

## 📌 Project Overview

Financial Statement Analysis is an important method used to evaluate the financial performance and stability of a company. This project focuses on analyzing a company’s financial statements using **Microsoft Excel** to understand its profitability, liquidity, and overall financial health.

The analysis is performed using key financial statements such as:

* Balance Sheet
* Income Statement
* Equity Statement

Various financial ratios and analytical techniques are applied to interpret the financial condition of the company and support better decision-making.

---

# 🎯 Objectives of the Project

* To understand the structure of financial statements.
* To analyze financial performance using **Excel formulas and functions**.
* To calculate important **financial ratios**.
* To compare financial data across different years.
* To present financial insights using **charts and visualizations**.

---

# 🧩 Project Modules

## 1️⃣ Financial Data Collection Module

This module collects financial information from company financial reports such as:

* Balance Sheet (Assets, Liabilities, Equity)
* Income Statement (Revenue, Expenses, Net Profit)
* Cash Flow Statement

The data is organized into Excel sheets for analysis.

---

## 2️⃣ Ratio Analysis Module

Key financial ratios are calculated to evaluate company performance.

### Liquidity Ratios

* Current Ratio
* Quick Ratio

### Profitability Ratios

* Net Profit Margin
* Return on Assets (ROA)

### Solvency Ratios

* Debt to Equity Ratio
* Interest Coverage Ratio

---

## 3️⃣ Comparative Analysis Module

### Horizontal Analysis

Compares financial performance between two years.

### Vertical Analysis

Represents financial statement items as percentages of a base value.

---

## 4️⃣ Visualization and Reporting Module

Excel tools are used to create graphical analysis including:

* Line charts
* Bar charts
* Pie charts
* Financial dashboards

These visualizations help in better understanding financial trends.

# 📊 Financial Formulas Used

### 1️⃣ Gross Profit Margin

```
(Sales - Cost of Goods Sold) / Sales
```

### 2️⃣ Operating Profit Margin

```
EBIT / Sales
```

### 3️⃣ Net Profit Margin

```
Net Income / Sales
```

### 4️⃣ Return on Assets (ROA)

```
Net Income / Total Assets
```

### 5️⃣ Return on Equity (ROE)

```
Net Income / Shareholders’ Equity
```
